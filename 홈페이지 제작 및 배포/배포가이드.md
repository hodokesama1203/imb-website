# IMB 홈페이지 배포 가이드 (www.ufb.co.kr)

## 폴더 구조

```
IMB_홈페이지/
├── index.html          ← 메인(홈)
├── about/index.html    ← 회사소개
├── business/index.html ← 산업용 소재 솔루션 + 기술 컨설팅
├── ufb/index.html      ← i-UFB 솔루션 + 제품 라인업
├── contact/index.html  ← 오시는 길 · 문의 (지도 포함)
└── assets/img/         ← 로고 및 제품 사진 (웹 최적화 완료)
```

미리보기: `index.html`을 더블클릭하면 브라우저에서 바로 열립니다. (인터넷 연결 필요 — 폰트/아이콘 CDN 사용)

## 배포 절차 (Vercel 무료, 권장)

1. **GitHub 가입/로그인** → github.com
2. **새 저장소 생성** (예: `imb-website`, Public)
3. 저장소 페이지에서 **"uploading an existing file"** 클릭 → 이 폴더의 파일 전체를 드래그하여 업로드 → Commit
4. **Vercel 가입** → vercel.com (GitHub 계정으로 로그인)
5. **Add New → Project** → `imb-website` 저장소 Import → Deploy (설정 변경 없이 기본값)
6. 1분 내 `imb-website.vercel.app` 주소로 사이트 공개됨

## 도메인 연결 (ufb.co.kr)

1. Vercel 프로젝트 → **Settings → Domains** → `ufb.co.kr`, `www.ufb.co.kr` 추가
2. 도메인 구입처(가비아/후이즈 등) DNS 관리에서 Vercel이 안내하는 값 입력:
   - A 레코드: `@` → `76.76.21.21`
   - CNAME: `www` → `cname.vercel-dns.com`
3. 적용까지 보통 10분~수 시간. HTTPS는 자동 발급됩니다.

## 배포 후 할 일

- **네이버 서치어드바이저** (searchadvisor.naver.com): 사이트 등록 → 소유 확인 → 검색 노출 요청
- **구글 서치콘솔** (search.google.com/search-console): 동일하게 등록
- 카톡/문자 공유 시 미리보기(OG 이미지)는 이미 설정되어 있음

## 수정이 필요할 때

- 텍스트 수정: 각 `index.html`을 메모장/VS Code로 열어 해당 문구 수정 → GitHub에 다시 업로드하면 자동 재배포
- 사진 교체: `assets/img/` 안의 파일을 같은 이름으로 교체

## 비용

- 도메인 갱신비만 발생 (연 1~2만 원, 기존 보유)
